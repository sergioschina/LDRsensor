# LDRsensor

## Summary
Implementation of a brightness sensor to turn on a light using basic components and assembly code

### Hardware:
- Microcontroller Atmega328p
- Light Sensor (LDR)
- Lamp / LED

### Software:
- [Proteus](https://www.labcenter.com/) 
- avrdudess 2.4 (avrdude version 6.1)

### Language:
- Assembly
